PrivacyScore is released as open source software. All collected datasets will be archived and published on the website.

Source Code

The whole source code necessary to run an instance of PrivacyScore is available at GitHub. The code is licensed under the GPLv3+.

Open Data

We will publish the collected scan results of all lists on the PrivacyScore website (with the exception of lists that are marked as private). Furthermore, the results can be downloaded in machine-readable form.
